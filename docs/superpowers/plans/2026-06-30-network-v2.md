# Network v2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Upgrade the binary-reward dashboard network into an interactive, mechanic-explaining, settlement-dramatizing canvas (upline highlight, L/R + volume edge encoding, richer nodes + hover inspector, faithful settlement animation, zoom/pan/collapse) backed by one read-only settlement endpoint.

**Architecture:** One new backend read endpoint (`GET /settlement/latest`) exposes the latest batch's PairingRecords. The frontend keeps `layoutTree` pure (collapse = caller filters its input; edges gain `position`+`weight`), holds all interaction state (zoom/pan/collapse/highlight) in `TreeCanvas`, and drives a faithful GSAP settlement timeline from the endpoint. Vitest + RTL cover pure logic, components, and timeline construction; visual verification via screenshots.

**Tech Stack:** NestJS + Mongoose (backend), Next.js 14 + React + Tailwind + GSAP (frontend), Vitest + RTL, Playwright.

## Global Constraints

- Node >= 20. Comments in English; explanations to user in Simplified Chinese.
- No new dependencies (zoom/pan/collapse are hand-rolled). No `: any` anywhere — Next lint fails the build on `@typescript-eslint/no-explicit-any`.
- Money/volume render with monospace tabular numerals (`.mono-num`). Palette: gold `#f5b233` = money, cyan `#38e1ff` = left-leg volume, teal `#5eead4` = right-leg volume, emerald `#34d399` = bonus/pairing, amber `#f59e0b` = cap-hit, rose `#fb7185` = error.
- All GSAP runs inside a `useGSAP` scope and honors `prefers-reduced-motion` via `gsap.matchMedia()`.
- `npm run build` (frontend) must stay clean; backend `npx jest` and frontend `npx vitest run` must stay green.
- Preserve existing contracts: tree node cards keep `data-node-id={node.id}`; edges keep `data-edge="${fromId}->${toId}"`; `onSelect(node.userId)`; panel prop signatures; `useDashboard` return shape is additive only.
- Backend runs on :3100, frontend on :3000 (Mongo + Redis already running). Do not commit unless authorized.

```ts
async latest(): Promise<LatestSettlement> {
  const [batch] = await this.batchModel
    .find({ status: 'completed' })
    .sort({ endedAt: -1, createdAt: -1 })
    .limit(1)
    .lean()
    .exec();
  if (!batch) {
    return { batchId: '', triggeredBy: 'manual', totalBonus: 0, endedAt: null, records: [] };
  }
  const raw = await this.recordModel.find({ batchId: String(batch._id) }).lean().exec();
  const records: SettlementRecord[] = raw.map((r: any) => ({
    nodeId: r.nodeId, pairedAmount: r.pairedAmount, bonus: r.bonus,
    cappedAmount: r.cappedAmount, carryLeftAfter: r.carryLeftAfter, carryRightAfter: r.carryRightAfter,
  }));
  return {
    batchId: String(batch._id), triggeredBy: batch.triggeredBy,
    totalBonus: batch.totalBonus, endedAt: batch.endedAt ?? null, records,
  };
}
```

```tsx
'use client';
import type { Edge } from '@/lib/useTreeLayout';

export function TreeEdges({ edges, width, height, highlightedKeys }:
  { edges: Edge[]; width: number; height: number; highlightedKeys?: Set<string> }) {
  const hasHighlight = !!highlightedKeys && highlightedKeys.size > 0;
  return (
    <svg width={width} height={height} className="pointer-events-none absolute left-0 top-0 overflow-visible">
      {edges.map((e) => {
        const key = `${e.fromId}->${e.toId}`;
        const on = !hasHighlight || highlightedKeys!.has(key);
        const color = e.position === 'L' ? '#38e1ff' : '#5eead4';
        return (
          <path
            key={key}
            data-edge={key}
            d={e.d}
            fill="none"
            stroke={color}
            strokeWidth={1 + e.weight * 2.5}
            strokeLinecap="round"
            opacity={on ? 0.55 + e.weight * 0.35 : 0.08}
          />
        );
      })}
    </svg>
  );
}
```

```ts
import { gsap } from 'gsap';
import type { SettlementRecord } from '@/lib/types';

// Faithful settlement: pulse each node that earned a bonus this batch.
// jsdom-safe — records with no matching card are skipped. Caller gates on reduced-motion.
export function settlementSequence(root: HTMLElement, records: SettlementRecord[]): gsap.core.Timeline {
  const tl = gsap.timeline({ paused: true });
  records.filter((r) => r.bonus > 0).forEach((r, i) => {
    const card = root.querySelector<HTMLElement>(`[data-node-id="${r.nodeId}"]`);
    if (!card) return;
    tl.fromTo(
      card,
      { boxShadow: '0 0 0px rgba(52,211,153,0)' },
      { boxShadow: '0 0 22px rgba(52,211,153,0.8)', duration: 0.32, yoyo: true, repeat: 1 },
      i * 0.12,
    );
  });
  tl.play();
  return tl;
}
```
